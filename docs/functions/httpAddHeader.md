# $httpAddHeader
> Adds an HTTP header
## Usage
```
$httpAddHeader[name;...value]
```
| Name | Type | Description | Required | Spread
| :---: | :---: | :---: | :---: | :---: |
name | String | The header name | Yes | No
value | String | The header value | Yes | Yes
