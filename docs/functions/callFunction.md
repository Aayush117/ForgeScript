# $callFunction
> Calls a forge function made by the user.
## Usage
```
$callFunction[name;...args]
```
| Name | Type | Description | Required | Spread
| :---: | :---: | :---: | :---: | :---: |
name | String | The function name | Yes | No
args | String | The args to call this function with | Yes | Yes
